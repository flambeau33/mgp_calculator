# Calculateur de MGP en C

Ce programme calcule la moyenne génerale pondérée (MGP) en se basant sur:
-la note obtenue /100
-le nombre de crédits de l'UE

Il assigne automatiquement :
-le grade obtenue(A,B,B-,C+...) en fonction de la note entrée 
-la valeur de la mgp obtenue pour chaque notes

# Compilation
gcc mgp_calculator.c -o mgp

# Execution
./mgp
